# 📝 版本變更記錄 (CHANGELOG)

本文件記錄「書籤檢查小幫手」的所有版本更新與改進內容。

---

## 🚀 [v1.1.0] - 2026-05-18

這是一次重大的**功能優化與介面美化更新**！在此版本中，我們針對書籤檢查結果的二次過濾、失效連結的前置多選篩選，以及極致的 UI 視覺細節（數量紅色顯示、長路徑絕不折行、高對比標籤）進行了全面的重構與升級。

### 🆕 新增功能 (Features)
- **重複書籤「結果內過濾與隱藏」**：
  - 工具列新增過濾選單，點選特定資料夾後，畫面僅保留該資料夾下的重複書籤，其他無涉項目自動隱藏，確保界面純淨。
  - **全選防呆安全機制**：套用過濾時，「全選建議刪除項目」只會勾選當前**畫面上可見**的重複書籤，避免隱藏在背後的其他項目被誤勾刪除。
- **失效連結「檢查前資料夾多選篩選」**：
  - 引入客製化彈出式多選下拉面板（`custom-multiselect`），支援對任意多個目標資料夾同時進行精準掃描。
  - 支援 **Unicode 層次化縮排** 展示書籤樹結構、**即時關鍵字搜尋** 過濾列表，以及 **一鍵全選 / 一鍵清除** 捷徑。
- **異常狀態「前置高對比實心 Badge」**：
  - 檢測結果的標題最前方置入醒目的實心 Badge，秒速判讀錯誤類型（⏱️ 逾時、❌ 錯誤狀態碼、↪️ 重新導向），極大地提升了失效原因的可讀性。

### 🎨 介面與體驗優化 (Improvements)
- **結果二次過濾「高質感客製化單選選單」**：
  - 將重複書籤與失效連結結果上方的過濾 `select` 升級為客製化單選下拉元件，突破原生選單無法局部上色的限制。
  - **紅色異常計數標章**：在資料夾選項右側，以醒目的**紅色粗體 `(N)`** 標明該資料夾下的異常數量。
  - **自適應超寬撐開（絕不折行）**：下拉清單最大寬度統一擴展至 **`480px`**，結合 **`width: max-content;`**，即使資料夾階層再深，文字也絕不折行。
  - **超高縱深可視長度**：將下拉選單最大高度加長至 **`380px`**，一眼可完整瀏覽 10~12 個以上選項，減少了 60% 的滾動滑鼠次數。
  - **實心不透明主題背景**：背景指定為實心主題色 `var(--bg-surface)`，在深色與亮色主題下都有完美的遮罩遮擋效果，可讀性高達 100%。

## 📅 [v1.0.0] - 2026-05-15

### 🎉 初始發佈
- 提供重複書籤檢測功能（支援 URL、網域、根網域比對模式）。
- 提供失效連結檢測功能（實作五層降級智慧探測 Service Worker）。
- 實作安全刪除與回收站機制，支援一鍵復原與自動清理。
- 支援 JSON 與 HTML (Netscape 格式) 雙格式匯出與拖曳匯入。
- 支援設定面板，自由調整並發數、超時秒數與清理規則。
- 精美的現代化暗黑設計系統，高對比度美學與Toast即時通知。
